# Tutorial on the Visual Behavior Neuropixels dataset

This tutorial is synced with https://github.com/AllenNeuralDynamics/vbn_tutorial

There are two nearly identical notebooks:

- vbn_tutorial_CO.ipynb is the notebook set up for running in Code Ocean, where the environment has already been set and the data is attached to the capsule

- vbn_tutorial_local.ipynb is the notebook set up to be run locally where the data gets downloaded and stored to a specified location.
